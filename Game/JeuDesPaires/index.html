Comment jouer ?

Les cartes sont retournées face cachées, afin de gagner la partie vous disposez de 45 secondes pour débusquer toutes les cartes !

Pour cela, retournez une carte puis une deuxième :
- si les cartes sont identiques elles sont retiré du jeu
- si les cartes sont différentes elles se retournent et reste dans le jeu au même endroit

Lorsque le jeu ne contient plus de carte ou que le temps s'est écoulé la partie est terminée.
